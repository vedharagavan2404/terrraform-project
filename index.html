AWS DevOps Projects

Project 1: AWS Infrastructure Automation

This project involved automating the provisioning and management of AWS resources using Infrastructure as Code (IaC) tools such as AWS CloudFormation and Terraform. I designed and implemented templates to create and manage EC2 instances, VPCs, S3 buckets, and more. This automation reduced manual configuration errors and improved resource scalability.

Project 2: Continuous Integration and Deployment (CI/CD)

In this project, I set up a CI/CD pipeline using AWS CodePipeline and AWS CodeBuild. The pipeline automated the testing, building, and deployment of applications hosted on AWS Elastic Beanstalk. I integrated GitHub with the pipeline, enabling automatic deployments upon code commits. This streamlined the development workflow and ensured consistent deployments.
